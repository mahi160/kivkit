# Kivkit
A shadcn like component and widget library. 

## What?
- A cli to copy code from registry
- Primitive component for different frontend libraries
  - React: Base UI
  - Svelte: Bits UI
  - Angular: Angular Material (?)
  - Vue: Reka UI
  - Web Components: Lit?
- Specialized widgets: Text editor, map, charts(?) etc.
## Why
- All ui is shadcn look alike
- Learn modern css.Target *baseline newly available*
- Non dependent on tailwind css
- Easily customizable themes
